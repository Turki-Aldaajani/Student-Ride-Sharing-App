# Student Ride-Sharing App

## Project Overview
Student Ride-Sharing App is a university-based ride coordination system designed to help students create, search, and request rides securely within the campus community. The system improves transportation efficiency and supports safe ride-sharing within the university environment.

## Implemented and Tested Features
The following features have been implemented and tested based on Sprint 1, Sprint 2, and Sprint 3:

- [x] User Registration & Login
- [x] Verify Student ID
- [x] Create Ride
- [x] Request Ride
- [x] Driver Matching
- [x] Ride Completion
- [x] Payment Processing
- [x] Rating & Feedback

## Testing
All listed features were tested using manual functional testing derived from system use cases.  
Detailed test cases and results are documented in the Test Plan.

## Tech Stack
- Platform: Web / Mobile Application
- External Services: Payment Gateway, Map/Geo-location Service
